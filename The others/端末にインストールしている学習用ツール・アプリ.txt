端末にインストールしている学習用ツール・アプリ（2021/10）
自分自身の主な用途について
使用しているOSはWin10


・VMware Workstation 16 Player、Oracle VM VirtualBox
ローカルに仮想マシンを構築して実際に触りながら学習する。

・Teraterm、Putty
ターミナルソフト。ローカルの仮想マシンやクラウド上の仮想マシンに
SSH接続する際に使用することがほとんど。

・Wireshark
仮想マシンなどに接続できないときに、パケットキャプチャして
通信の様子を確認する。

・Visual Studio Code
CloudFormationなどのコーディング用に使用。

・WindowsTerminal
コマンドプロンプトやPowerShellを使用するときに便利。

・メモ帳
メモ用によく使用する。なんでもメモする。

・GitBash
クラウド上などの仮想マシンにssh接続するときのsshキーを生成したり、
ローカル内で移動させたりするときに使用する。

・アタッシェケース4
ファイルなどを暗号化するときに使用する。

・7-Zip File Manager
ファイルやフォルダの圧縮や解凍にしようする。

・SoftEther VPN Client Manager
ダウンロードに不安があるときに使う。
外出先で公共Wi-Fiに接続しなければならないときに使用。

・Chrome、Firefox
クラウド使用時に使うブラウザ。Web/ミドルウェアの表示にもよく使う。

・AWS CLI、GCP SDK
ローカルからクラウドにログインしたり、操作したりしたいときに使用する。

・Noxプレイヤー
アンドロイド端末の仮想化。
アンドロイド端末のアプリと同期させて使用。

・MSオフィス（Word、Excel、PowerPoint）
構成図や資格学習、手順書作成などに使用する。

・Snip & Sketch
スクショ撮る用

・PowerAutomateDesktop
自動化ツール。自動化の学習用。

・MEGAsync
ストレージ。

・WinMerge
ログやコンフィグの差分表示に使用。
