VMware Workstation Player16にてUbuntu Serverをインストールし、
Win10端末のターミナルソフトからSSH接続する。

1.Ubuntu 18.04 LTS Server をインストールする。
isoファイルのダウンロードは割愛

1.1.インストールメディアからisoファイルを起動し、
各種インストール前の設定を進めていく。

1.2.言語選択
Englishを選択

1.3.Keyboard configuration
layoutとvariantともにJapaneseに設定

1.4.Network Connection 
手動設定の場合は、アダプタを選択し、ipアドレスやサブネット、GW
を指定。

Subnet:192.168.0.0/24
IP:192.168.0.111
GW:192.168.0.254
DNS:8.8.8.8

1.5.Configure Proxy
プロキシの設定
設定しない場合は→done

1.6.Ubuntu Archive Mirror 
日本のサーバーが選択されていることを確認する。

1.7.FileSystem setup
Use an entire disk

1.8.choose the disk to install to
インストール先のディスクを選択

1.9.Filesystem summary 
内容を確認し、done

1.10.profile setup
your name 
your server's name 
pick a username 
choose a password 
confirm your password 

1.11.ssh setup 
openssh server 
の[]にspaceキーを押し、[X]の状態にする。
選択したら、done

1.12.featured server snaps
必要なソフトウェアがあれば選択する。
必要なければ、done

1.13.installing system
cancel update and reboot でアップデートのインストールを中止にできる
インストール後、rebootを選択

1.14.再起動後、usernameとパスワードを入力し、ログイン

1.15. sudo apt update && sudo apt upgrade -y でアップデート

1.16.sshサーバーの軌道を確認
sudo systemctl status sshd 
sshサーバーがactiveになっていることを確認する。

1.17.端末から仮想マシンに対してssh接続
win10端末のipアドレスを192.168.0.0/24のセグメントに設定し、
teratermやputtyなどのターミナルソフトを使用する。サーバーの
ipアドレス192.168.0.111を入力し、接続する。
（サーバーのユーザー名とパスワードが必要）
